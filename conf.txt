les fonctions du module : 

TBGetProfiles [-profilesFolder chemin_vers_le_repertoire_profiles]Récupère la liste des profils Thunderbird sous la forme d'un tableau de chaînes comportant les chemins vers les profils.-profilesFolder peut être défini pour pointer sur un chemin personnalisé. TBGetConfig -profilePath <chemin_du_profil>Récupère le contenu du fichier prefs.js d'un profil sous la forme d'une chaîne de caractères. TBSaveConfig -profilePath <chemin_du_profil> -configuration <conf>Sauvegarde la conf fournie pour le profil indiqué. Créer un backup avant de faire la sauvegarde si une conf est déjà présente.-configuration doit être de type array-profilePath doit être une chaîne de caractères TBCreateConfigBackup -profilePath <chemin_du_profil>Permet de faire un backup incrémentiel au format prefs-#.js où # est un nombre incrémenté de la conf actuelle du profil indiqué. TBGetMailAccountSettings -configuration <conf> [-id id_du_compte]Permet de récupérer sous la forme d'une hashtable l'ensemble de la conf des comptes mails présents dans la conf.Il est possible de préciser un id avec le paramètre -id pour ne récupérer que la conf d'un compte précis. TBSetMailAccountSettings -configuration <conf> -id <id> -settings <hashtable>Permet de définir des paramètres de conf (ajout, modification) pour un compte mail précis.Les paramètres doivent être fournis sous la forme d'une hashtable qui suit le modèle suivant :@{ "useremail" = "ce.dsi@ac-nancy-metz.fr"; "sig_file" = "Chemin_vers_la_signature"; } TBSetGlobalSettings -configuration <conf> -settings <hashtable>Permet de définir des paramètres de conf (ajout, modification) dans la conf.Les paramètres doivent être fournis sous la forme d'une hashtable qui suit le modèle suivant :@{ "network.proxy.type" = 4; "mail.identity.id1.useremail" = "Chemin_vers_la_signature"; } TBDeleteMailAccountSettings -configuration <conf> -id <id> -settings <array>Permet de supprimer un paramètre de la conf pour un compte de messagerie.Les paramètres doivent être fourni sous la forme d'un array qui suit le modèle suivant :@("useremail", "sig_file") TBDeleteGlobalSettings -configuration <conf> -settings <array>Permet de supprimer un paramètre dans la conf.Les paramètres doivent être fourni sous la forme d'un array qui suit le modèle suivant :@("network.proxy.type", "mail.identity.id1.useremail")
